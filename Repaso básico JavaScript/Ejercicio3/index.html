Propiedades y métodos básicos

  1. innerHTML, textContent:

    2. Contenido HTML o texto dentro del elemento

  2. getAttribute(nombre):

    3. Obtiene el valor de un atributo del elemento

  3. setAttribute(nombre, valor):

    4. Establece el valor de un atributo del elemento

  4. classList:

    5. Lista de clases CSS aplicadas al elemento.

  5. addEventListener(evento, función):

    6. añade un listenerm una funcion
